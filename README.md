# I2A2 |  Projeto de Conclusão do Curso

Repositório destinado ao trabalho final do Curso de Agentes Autônomos com Redes Generativas da I2A2.

Os seguintes ``diretórios`` são encontrados:

- **apresentacao_pitch_deck_pptx**: Diretório contendo arquivo .pptx da apresentração pitch deck
- **relatorio_pdf**:  Diretório contendo arquivo .pdf contendo o relatório do projeto 
- **tour-extraction-system**: Diretório referente ao sistema desenvolvido para Extração de Tarifários de Turismo em PDF
- **video_demo_instalacao_e_execucao_sistema_mp4**: Diretório contendo os links para vídeos no Youtube com demonstração de configuração e execução do sistema 
- **video_pitch_deck_mp4**: Diretório contendo arquivo .mp4 com a apresentação do pitch deck
